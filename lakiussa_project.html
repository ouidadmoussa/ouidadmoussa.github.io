Overview

Lakiussa is a business-to-business web application that helps suppliers to expose their detailed information and services or products. On other hands clients can compare easily different suppliers offers and give their feedback about any company.

Challenge

The main goal and challenge on this project were to display a large amount of information in a simple and clear way. build a responsive, fast and flexible application with a code that humans can understand easily.

Solutions

In purpose to facilitate working process I built backend service using Node.js , Express and MongoDB, for the front End part I used Angular , Bootstrap 3 and HTML.
