Overview

This project was an amazing-to-work-on project. it is a SAAS and business-to-customer application that help business to get found by more customers.

How it works

  • the user creates a profile on the platform after subscribing
  • he customize his website than add products.
  • besides the website, those products gets added automatically to the platform so they will displayed easily to the customers.
  • the user also can share articles or images with customers on his profile
